# Code of Conduct

The project-template project adopts the
[**Contributor Covenant, version 2.1**](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
as its code of conduct. Participation in this project — as a contributor,
a reviewer, a maintainer, or a participant in the issue tracker or
discussions — means agreeing to the expectations laid out at the link
above.

The authoritative text is maintained at contributor-covenant.org and is
not reproduced here; please read it at the link above rather than relying
on any partial summary.

## Enforcement

Reports of conduct violations should reach the maintainers privately via
GitHub's private vulnerability reporting on this repository (the channel
also accepts conduct reports — title the report clearly as a conduct
concern).

Please **do not open public issues** for personal conduct complaints.
Maintainers will acknowledge reports within seven days and will keep the
reporter's identity confidential to the extent consistent with acting on
the report.

## Scope

This code of conduct applies to every project space — the issue tracker,
pull requests, discussions, documentation site, and any ancillary channel
operated on the project's behalf.
